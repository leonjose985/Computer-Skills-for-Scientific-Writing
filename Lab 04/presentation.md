# Цель работы

## Основная цель

Ознакомиться с темой целочисленной арифметики многократной точности, используя концепции, представленные в предыдущих работах, такие как модули, наибольший общий делитель и шифрование.

Задачи:

- Реализовать рассмотренные алгоритмы программно

# Выполнение лабораторной работы

## Алгоритм 1: сложение неотрицательных целых чисел

Векторы `u` и `v` представляют цифры чисел в базе `b`. Массив `w` инициализируется нулями с `n+1` элементами для хранения результата, включая перенос. Цикл перебирает цифры от наименее значимой до наиболее значимой.

![Код алгоритма сложения](image/01.png){ width=70% }

## Результат алгоритма сложения

Алгоритм корректно вычисляет сумму цифр из `u` и `v` в каждой позиции, включая перенос `k`, и сохраняет результат в массиве `w`.

![Результат выполнения алгоритма сложения](image/02.png){ width=70% }

# Алгоритм вычитания

## Алгоритм 2: вычитание неотрицательных целых чисел

Алгоритм вычитания обрабатывает заём между разрядами. При отрицательном результате в позиции `j` к разности прибавляется основание `b`, а переменная заёма `k` устанавливается в `-1`.

![Код алгоритма вычитания](image/03.png){ width=70% }

## Результат алгоритма вычитания

Алгоритм корректно вычисляет разность многоразрядных чисел с учётом заёмов между позициями.

![Результат выполнения алгоритма вычитания](image/04.png){ width=70% }

# Алгоритмы умножения

## Алгоритм 3: умножение неотрицательных целых чисел

Алгоритм перебирает цифры `v` и для каждой ненулевой цифры выполняет умножение на все цифры `u`, накапливая результат в массиве `w` с учётом переносов.

![Код алгоритма умножения](image/05.png){ width=70% }

## Алгоритм 4: быстрый столбик

Алгоритм быстрого столбика вычисляет произведение через суммирование по диагоналям, обрабатывая каждую позицию результата за один проход с промежуточной суммой `t`.

![Код алгоритма быстрого столбика](image/06.png){ width=70% }

# Алгоритм деления

## Алгоритм 5: деление многоразрядных целых чисел

Алгоритм деления включает предварительную корректировку остатка, основной цикл вычисления частного и вычитание с масштабированием. Используются вспомогательные функции `compare_large_integers`, `subtract_scaled` и `add_scaled`.

![Код алгоритма деления](image/07.png){ width=70% }

## Результат алгоритма деления

Алгоритм возвращает массив частного `q` и массив остатка `r`, корректно обрабатывая все случаи многоразрядного деления.

![Результат выполнения алгоритма деления](image/08.png){ width=70% }

# Итоги работы

## Вывод

Лабораторная работа успешно выполнена. В ходе работы были реализованы и освоены:
