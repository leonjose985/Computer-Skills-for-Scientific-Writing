2. Создание первого LaTeX-документа

Был создан файл формата .tex, содержащий базовую структуру документа LaTeX. В документе были использованы основные элементы:

объявление класса документа (article);


3. Добавление комментариев и структурирование документа

Задание: Добавить комментарии в документ, использовать параметры класса документа, добавить параграфы и примечание.

Выполнение:
Документ был расширен:

```latex

This is a new paragraph.
\end{document}
```
Были использованы:

Параметры класса документа: a4paper,12pt.

Освоение базовых принципов LaTeX является фундаментом для дальнейшего изучения более сложных возможностей системы.
