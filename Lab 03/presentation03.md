# Вводная часть

## Актуальность

Набор математических формул в LaTeX является ключевым навыком для исследователей, так как позволяет создавать профессионально оформленные научные публикации с точным воспроизведением математических выражений.

## Объект и предмет исследования

- Математический режим LaTeX (inline и display)
- Пакеты `amsmath`, `bm`, `mathtools`
- Форматирование формул: индексы, интегралы, матрицы, выравнивание

## Цели и задачи

Освоить набор математических формул в LaTeX с использованием математического режима, специализированных пакетов и инструментов форматирования.

## Материалы и методы

- Дистрибутив TeX Live
- Пакеты `amsmath`, `bm`, `mathtools`
- Команда `pdflatex`

# Результаты

## Математический режим: inline и display
```latex
% Встроенный режим
Формула в тексте: \( y = mx + c \), также \( 5^2 = 3^2 + 4^2 \)

% Выносный режим
\[
\int_{-\infty}^{\infty} e^{-x^2} \, dx
\]
```

## Индексы и специальные математические команды
```latex
% Индексы
\( a^{2} + b_{n} = c^{k}_{i} \)

% Греческие буквы и функции
\( \sin \theta + \log \alpha = \beta \)

% Дифференциал с пользовательской командой
\newcommand{\diff}{\mathop{}\!\mathrm{d}}
\[
\int_{0}^{1} x^2 \diff x
\]
```

## Выравнивание уравнений с amsmath
```latex
\begin{align*}
Q_{n,0} &= 1, \quad Q_{0,k} = [k=0]; \\
Q_{n,k} &= Q_{n-1,k} + Q_{n-1,k-1} + \binom{n}{k}, \\
        &\quad \text{для } n,k > 0.
\end{align*}
```

## Шрифты и жирные символы в математике
```latex
% Разные шрифты
\( \mathrm{Roman}, \mathit{Italic}, \mathbf{Bold}, \mathbb{Blackboard} \)

% Жирные символы с пакетом bm
\( \bm{\alpha} + \bm{\beta} = \bm{\gamma} \)

% Правильное использование
\( \mathit{size} \)
```
