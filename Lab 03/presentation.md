# Цель работы

## Основная цель

Освоить набор математических формул в LaTeX с использованием математического режима (inline и display), пакетов **amsmath**, **bm**, **mathtools**, а также форматирования формул: индексы, интегралы, матрицы и выравнивание.

Задачи:

- Изучить разницу между встроенным и выносным математическим режимом
- Освоить команды для специальных символов и греческих букв
- Научиться выравнивать уравнения с помощью `align`
- Работать с математическими шрифтами и жирными символами
- Создавать нумерованные уравнения и матрицы

# Выполнение лабораторной работы

## Математический режим: inline и display

Были изучены два режима набора формул: встроенный `\( ... \)` для формул внутри текста и выносной `\[ ... \]` для отдельных формул, таких как интеграл Гаусса.

![Пример inline и display формул](image/01.png){ width=70% }

## Результат математического режима

В PDF-документе корректно отображаются как встроенные формулы, так и выносные, с правильными отступами и размерами символов.

![Результат отображения формул в PDF](image/02.png){ width=70% }

# Выравнивание уравнений

## Окружение align с пакетом amsmath

Было изучено окружение `align*` для многострочных уравнений. Использовались:

- `&` для задания точки выравнивания;
- `\\` для разделения строк;
- `\quad` для горизонтального пробела;
- `\text{}` для вставки обычного текста внутри формул.

![Выравнивание уравнений с align*](image/05.png){ width=70% }

## Результат выравнивания

В PDF-документе многострочные уравнения выровнены по знаку равенства с корректным отображением всех элементов.

# Математические шрифты

## Шрифты и жирные символы

Были изучены различные математические шрифты и применение пакета **bm** для жирных символов:

- `\mathrm{}` — Roman;
- `\mathit{}` — Italic;
- `\mathbf{}` — Bold;
- `\mathbb{}` — Blackboard;
- `\bm{}` — жирные греческие символы.

![Математические шрифты в редакторе](image/07.png){ width=70% }
