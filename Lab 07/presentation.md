# Цель работы

## Основная цель

Освоить создание презентаций и научных постеров в LaTeX с использованием класса **Beamer** для презентаций и пакетов **a0poster**, **beamerposter**, **tikzposter** для постеров.

Задачи:

- Изучить структуру презентаций Beamer
- Освоить управление появлением контента (`\pause` / `\uncover`)
- Научиться создавать многостолбцовые макеты постеров
- Сравнить три метода создания постеров

# Выполнение лабораторной работы

## Базовая структура презентации Beamer

Была создана презентация с темой `Copenhagen`, включающая заголовочный слайд и содержательные слайды. Структура документа строится на окружении `\begin{frame}...\end{frame}` для каждого слайда.

# Управление контентом

## Команды \pause и \uncover

Были изучены инструменты пошагового отображения контента. Команда `\uncover<n->{}` позволяет показывать элементы начиная с указанного шага, создавая динамическую анимацию слайда.

![Управление появлением контента в редакторе](image/05.png){ width=70% }

## Результат пошагового отображения

Слайды с `\uncover` генерируют несколько версий одного кадра, каждая из которых последовательно раскрывает дополнительный контент.

![Результат пошагового отображения в PDF](image/06.png){ width=70% }
