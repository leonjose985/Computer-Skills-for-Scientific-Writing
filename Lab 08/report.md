# Цель работы

Изучить возможности пакета **TikZ** для создания графических объектов непосредственно в среде LaTeX. В рамках работы рассматриваются построение графов с нумерованными рёбрами, визуализация математических функций на координатной плоскости, а также генерация фрактальных структур с применением итеративных конструкций, узлов, путей и параметрических кривых.

# Ход выполнения

## Построение графа: задание *Exercise 8.2.1*

Для выполнения первого задания был подготовлен исходный файл `exercise8_1.tex` с описанием графа средствами пакета **TikZ**. Компиляция выполнена командой `pdflatex` в среде **TeX Live 2025**. В ходе сборки подключался графический стек `pgf` и класс документа `article`. Сообщение `No file exercise8_1.aux` является стандартным для первого прохода компиляции.

Вывод компилятора на начальном этапе сборки файла показан на скриншоте:

![Начало компиляции exercise8_1.tex](image/01.png){ #fig:001 width=70% }

Компиляция завершилась без ошибок: был сформирован файл `exercise8_1.pdf` объёмом 24043 байта и создан журнал `exercise8_1.log`. Завершающая часть вывода компилятора приведена на скриншоте:

![Завершение компиляции exercise8_1.tex](image/02.png){ #fig:002 width=70% }

В итоговом PDF-документе сформирован взвешенный граф с семью вершинами. Внешние вершины **A–E** образуют пятиугольную структуру, дополненную внутренним узлом **F** и центральным узлом **Z**. Все узлы оформлены окружностями, рёбра соединены командой `\draw`, числовые веса размещены вдоль линий в виде подписанных узлов.

![Результат Exercise 8.2.1: граф с взвешенными рёбрами](image/03.png){ #fig:003 width=70% }

## Построение графика функций: задание *Exercise 8.2.2*

На следующем этапе был скомпилирован файл `exercise8_2.tex`, содержащий описание координатного графика. Компилятор использовал тот же дистрибутив **TeX Live 2025** и загружал вспомогательные пакеты графического стека PGF/TikZ, обеспечивающие корректное построение кривых.

Начальный этап компиляции файла `exercise8_2.tex` показан на скриншоте:

![Начало компиляции exercise8_2.tex](image/04.png){ #fig:004 width=70% }

Сборка завершилась успешно: сформирован файл `exercise8_2.pdf` объёмом 40935 байт. Завершение компиляции подтверждается скриншотом:

![Завершение компиляции exercise8_2.tex](image/05.png){ #fig:005 width=70% }

Полученный PDF-документ содержит координатный график с двумя функциями — **y = eˣ** и **y = ln(x)** — и вспомогательными линиями **x = 1** и **y = 1**. Каждая кривая выделена отдельным цветом и снабжена подписью непосредственно на поле графика. Оси координат подписаны символами `$x$` и `$y$`, построение выполнено командой `plot` с параметрами `domain` и `samples`.

![Результат Exercise 8.2.2: график функций eˣ и ln(x)](image/06.png){ #fig:006 width=70% }

Завершающим заданием стала компиляция файла `exercise8_3.tex` с реализацией фрактальной структуры. В процессе сборки компилятор загружал модули PGF/TikZ, поддерживающие итеративные и рекурсивные конструкции, необходимые для генерации фрактала.

Начало компиляции файла `exercise8_3.tex` приведено на скриншоте:

![Начало компиляции exercise8_3.tex](image/07.png){ #fig:007 width=70% }

Компиляция завершилась без ошибок с формированием файла `exercise8_3.pdf` объёмом 20694 байта и журнала `exercise8_3.log`. Завершение сборки показано на скриншоте:

![Завершение компиляции exercise8_3.tex](image/08.png){ #fig:008 width=70% }

Структура построена на основе повторяющихся геометрических примитивов с применением масштабирования и симметрии, что наглядно демонстрирует возможности TikZ для создания сложных итеративных изображений.

# Выводы

В ходе выполнения лабораторной работы были изучены и практически отработаны следующие возможности пакета **TikZ**:

- подключение пакета `tikz` и организация графических объектов внутри среды `tikzpicture`;
- построение взвешенных графов с использованием узлов (`node`) и рёбер (`\draw`), включая размещение числовых подписей вдоль соединительных линий;
- визуализация математических функций с заданием области определения через параметр `domain`, управлением точностью через `samples`, добавлением осей, вспомогательных линий и текстовых подписей;
- применение цветового оформления для различения кривых на одном графике;
- реализация итеративных и рекурсивных структур на примере фрактала «ковёр Серпинского» с использованием масштабирования и геометрических примитивов;
- компиляция документов командой `pdflatex` и анализ логов сборки в среде **TeX Live 2025**.

Все три задания были успешно скомпилированы, полученные PDF-документы полностью соответствуют поставленным требованиям и наглядно иллюстрируют ключевые приёмы работы с графикой в LaTeX.
