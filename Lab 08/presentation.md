# Цель работы

## Основная цель

Освоить создание векторной графики и диаграмм в LaTeX с использованием пакета **TikZ** для программирования графики, а также рекурсивных функций и циклов для создания сложных фигур.

Задачи:

- Изучить базовые элементы TikZ: линии, узлы, координаты
- Освоить построение графиков математических функций
- Научиться использовать циклы для создания повторяющихся элементов
- Реализовать рекурсивные алгоритмы для фрактальных структур

# Выполнение лабораторной работы

## Создание графа с использованием TikZ

Был воспроизведён граф из шести узлов, расположенных по кругу. Узлы задаются командой `\node` с координатами в полярной форме, а рёбра — командой `\draw[->]` с метками на середине дуг.

![Код построения графа в редакторе](image/01.png){ width=70% }

## Результат построения графа

В PDF-документе граф отображается с шестью узлами, равномерно распределёнными по окружности, и направленными рёбрами с числовыми метками.

![Результат графа в PDF](image/02.png){ width=70% }

# Графики математических функций

## Построение координатных осей и функций

Были созданы координатные оси и графики нескольких функций с помощью команды `\draw` с параметром `domain` и `samples`. Использовались три кривые: экспоненциальная $y = e^x$, горизонтальная $y = 1$ и вертикальная $x = 1$.

![Код построения графиков функций](image/03.png){ width=70% }

## Результат графиков функций

В итоговом PDF-документе графики отображаются в цвете с подписями осей и кривых, корректно масштабированные в заданной области.

![Результат графиков функций в PDF](image/04.png){ width=70% }

# Фрактальные структуры

## Генерация ковра Серпинского

Был реализован рекурсивный алгоритм для создания ковра Серпинского с помощью `\tikzmath`. Алгоритм делит квадрат на 9 равных частей и рекурсивно заполняет 8 из них, пропуская центральный.

![Код рекурсивного алгоритма ковра Серпинского](image/05.png){ width=70% }

## Результат ковра Серпинского

В PDF-документе фрактальная структура корректно воспроизводится на заданной глубине рекурсии, демонстрируя самоподобие на каждом уровне.
