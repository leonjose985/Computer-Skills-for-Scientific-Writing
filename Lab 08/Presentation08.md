```latex

\draw[gray, ->] (-1,0) -- (3,0) node[right] {$x$};
\draw[gray, ->] (0,-1) -- (0,4) node[above] {$y$};
```

## 3. Генерация ковра Серпинского
Реализация рекурсивного алгоритма для создания фрактальной структуры:

```latex
\tikzmath{
    function sierpinskiCarpet(\x, \y, \s, \d) {
        if (\d == 0) then {
            { \fill (\x,\y) rectangle (\x+\s, \y+\s); };
        } else {
            \ns = \s / 3;
            for \i in {0,1,2} {
                for \j in {0,1,2} {
                    if not ((\i == 1) && (\j == 1)) then {
                        sierpinskiCarpet(\x + \i*\ns, \y + \j*\ns, \ns, \d-1);
                    };
                };
            };
        };
    };
}
```
