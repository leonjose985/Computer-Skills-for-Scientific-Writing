# Цель работы

Изучить структуру и правильный способ создания таблиц в LaTeX, освоить различные типы выравнивания, объединения ячеек и форматирования таблиц для оформления научных работ.

# Ход выполнения

## Поведение таблицы при избыточном количестве элементов

В противоположность предыдущему случаю была проверена ситуация, когда строка таблицы содержит больше элементов, чем задано колонок. В этом случае компилятор LaTeX выдаёт ошибку `Extra alignment tab has been changed to \cr`, указывающую на несоответствие числа разделителей `&` объявленной структуре таблицы.

# Выводы

В ходе выполнения лабораторной работы были приобретены необходимые знания и навыки для создания и форматирования таблиц в LaTeX:

- создание базовых таблиц с использованием окружения `tabular` и задание типов колонок `l`, `c`, `r` и `p` для различных вариантов выравнивания;
- добавление горизонтальных и вертикальных линий с помощью команд `\hline` и символа `|` в объявлении колонок;
- понимание поведения компилятора при недостаточном количестве элементов в строке (пустые ячейки) и при избыточном (ошибка `Extra alignment tab has been changed to \cr`);
- использование команды `\multicolumn` для объединения ячеек по горизонтали при создании заголовков и специальных строк таблицы.

Полученные навыки являются необходимой основой для создания профессионально оформленных таблиц в научных работах формата `.tex`, обеспечивая точность структуры и единообразие визуального представления данных.
