# Цель работы

## Основная цель

Освоить основные принципы создания и форматирования таблиц в LaTeX, изучить различные типы выравнивания колонок, методы объединения ячеек и профессиональное оформление таблиц для научных работ.

Задачи:

- Изучить базовую структуру таблиц в LaTeX
- Освоить различные типы выравнивания колонок
- Изучить поведение системы при ошибках в таблицах
- Научиться объединять ячейки с помощью `\multicolumn`

# Обработка ошибок

## Сценарий 1: недостаточное количество элементов

При недостаточном количестве элементов в строке LaTeX оставляет соответствующие ячейки пустыми, не прерывая компиляцию.

![Таблица с недостаточным количеством элементов](image/05.png){ width=70% }

## Сценарий 2: избыточное количество элементов

При избыточном количестве элементов в строке LaTeX генерирует предупреждение, а лишние элементы выходят за границы таблицы.

![Таблица с избыточным количеством элементов](image/06.png){ width=70% }

# Объединение ячеек

## Использование \multicolumn

Освоение техники горизонтального объединения ячеек для создания сложных табличных структур. Команда `\multicolumn{n}{align}{text}` позволяет объединить `n` колонок с заданным выравниванием.

![Таблица с объединёнными ячейками в редакторе](image/07.png){ width=70% }

## Результат объединения ячеек

В PDF-документе корректно отображается заголовок, объединяющий все три колонки, и частичное объединение ячеек в строке данных.

![Результат таблицы с multicolumn в PDF](image/08.png){ width=70% }
