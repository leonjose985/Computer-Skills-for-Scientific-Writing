# Цель работы

## Основная цель

Освоить работу с библиографией в LaTeX, изучить создание и использование BibTeX-файлов, научиться применять различные стили цитирования с помощью пакетов **natbib** и **biblatex**.

Задачи:

- Изучить структуру BibTeX-файлов
- Освоить процессы компиляции для natbib и biblatex
- Научиться создавать новые библиографические записи
- Исследовать различные стили цитирования

# Выполнение лабораторной работы

## Сравнение natbib и biblatex

Были изучены два основных подхода к работе с библиографией в LaTeX. Пакет **natbib** использует команды `\citet{}` для текстовых цитат и `\citep{}` для скобочных, со стилем `plainnat` и командой `\bibliography{}`.

![Работа с пакетом natbib](image/01.png){ width=70% }

## Результат компиляции с natbib

В PDF-документе корректно отображаются текстовые и скобочные цитаты, а также автоматически сформированный список литературы.

![Результат документа с natbib в PDF](image/02.png){ width=70% }

# Библиографические записи

## Создание новых записей BibTeX

Была расширена база данных `learnlatex.bib` новыми записями типа `@article`. Каждая запись включает поля: `author`, `title`, `journal`, `year`, `volume`, `number`, `pages` и `doi`.

![Новая запись BibTeX в файле .bib](image/03.png){ width=70% }

## Результат использования новых записей

Новые библиографические записи корректно отображаются в списке литературы и доступны для цитирования в тексте документа.

![Результат новых записей в PDF](image/04.png){ width=70% }

# Обработка ошибок

## Отсутствующие ссылки

При цитировании несуществующей записи командой `\cite{Nonexistent2023}` система генерирует предупреждения при компиляции, а в тексте отображается метка в виде знаков вопроса.

![Предупреждения при компиляции с отсутствующей ссылкой](image/05.png){ width=70% }

## Результат обработки отсутствующих ссылок

В PDF-документе место цитаты занимает знак `[?]`, что позволяет легко обнаружить и исправить отсутствующие ссылки.

![Результат документа с отсутствующей ссылкой в PDF](image/06.png){ width=70% }

# Стили цитирования

## Числовой стиль в natbib и biblatex

Были исследованы числовые форматы цитирования в обоих подходах. В **natbib** используется опция `[numbers]`, в **biblatex** — `style=numeric`, результатом является нумерованная цитата вида `[1]`.

![Числовой стиль цитирования в редакторе](image/07.png){ width=70% }

## Результат числового стиля цитирования

В PDF-документе цитаты отображаются в числовом формате, а список литературы нумеруется автоматически в порядке цитирования.

![Результат числового стиля цитирования в PDF](image/08.png){ width=70% }

# Итоги работы

## Вывод

Лабораторная работа успешно выполнена. В ходе работы были освоены:

- два подхода к работе с библиографией: **natbib** и **biblatex**;

Полученные навыки являются необходимыми для подготовки качественных научных публикаций с профессионально оформленным списком литературы.
