...

## 4.1 Числовой стиль цитирования
Эксперименты с числовым форматом цитирования в обоих подходах.

```latex
\usepackage[numbers]{natbib}
...
\citep{Graham1995} % Результат: [1]
```

## 4.2 Числовой стиль цитирования
```latex
\usepackage[style=numeric]{biblatex}
...
\autocite{Thomas2008} % Результат: [2]
```
