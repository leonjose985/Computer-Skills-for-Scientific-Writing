# Цель работы

## Основная цель

Освоить базовые принципы работы с системой компьютерной вёрстки LaTeX и установить дистрибутив TeX Live.

Задачи:

- Изучить способы установки TeX Live на различных операционных системах
- Ознакомиться с основными командами LaTeX
- Научиться создавать и компилировать простой LaTeX-документ
- Освоить базовую структуру LaTeX-файла

# Выполнение лабораторной работы

## Установка TeX Live

Установка дистрибутива TeX Live была выполнена с использованием пакетного менеджера операционной системы.

Для Ubuntu использовалась команда `sudo apt install texlive-full`.

Проверка корректности установки была выполнена с помощью команды `pdflatex --version`.

![Проверка установки TeX Live](image/01.png){ width=70% }

## Результат установки

Результат подтвердил успешную установку и готовность среды к работе.

![Результат команды pdflatex --version](image/02.png){ width=70% }

# Создание первого LaTeX-документа

## Структура файла .tex

Был создан файл формата `.tex` с базовой структурой LaTeX-документа, включающей:

- класс документа `\documentclass{article}`;
- подключение пакетов `babel` и `amsmath`;
- команды `\title`, `\author`, `\date` и `\maketitle`.

![Создание файла laboratorio1.tex](image/03.png){ width=70% }

## Компиляция документа

Для компиляции LaTeX-документа использовалась команда `pdflatex laboratorio1.tex`.

В результате был создан файл `laboratorio1.pdf`, содержащий структурированный отчёт.

![Компиляция laboratorio1.tex](image/04.png){ width=70% }

## Результат компиляции

После компиляции был получен PDF-документ с корректным форматированием.

![Результат laboratorio1.pdf](image/05.png){ width=70% }

# Математические формулы

## Использование математического режима

В ходе лабораторной работы были изучены основные способы ввода математических формул в LaTeX:

- inline-формулы с использованием `$...$`;
- display-формулы с использованием `\[...\]`.

# Итоги работы

## Вывод

Лабораторная работа успешно выполнена. В ходе работы были освоены:

- установка и настройка дистрибутива TeX Live;
- создание LaTeX-документов;
- работа со структурой документа (`\documentclass`, `\section`, `\maketitle`);
